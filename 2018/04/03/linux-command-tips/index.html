linux命令备忘

Posted on 2018-04-03

记录一些容易忘记的命令，持续更新

ls

  • 改变目录显示颜色

    cp /etc/COLORS ~/.dir_colors修改dir_colors里面的DIR后面的参数

date命令

  • 将指定时间戳转换对应日期

    date -d@149845400

  • 将指定时间转换成时间戳

    date “+%s” -d”20170810 03:20”

make

  • 错误日志屏幕显示同时重定向到文件

    make -j8 2>&1 >/dev/null | tee error

unzip

  • 解压到指定目录

    unzip xx.zip -d targetdir

grep

  • 动态日志过滤，开启line-buffered

    tail -f /mydir/myfile | grep –line-buffered “searchterm”

  • 匹配hello的行后面/前面/前后2行

    grep -A2/B2/C2 -i “hello” logfile

find

  • 查找home除dut目录下的txt 和rmvb文件

    find /home -path /home/dut/ -prune -name “.txt” -o -name “.rmvb”;

  • 如果要忽略两个以上的路径如何处理？

    find /home/carryf ( -path “/home/carryf/astetc” -o -path “/home/carryf/etc” ) -prune -o -type f -print
    注意( 和) 前后都有空格。

数组

a=(“1” “2” “3”); for i in ${a[@]};do echo $i;done

并、交、差

  • 并

    cat a b | sort | uniq > c # c is a union b

  • 交

    cat a b | sort | uniq -d > c # c is a intersect b

  • 差

    cat a b b | sort | uniq -u > c # c is set difference a - b

eval

  • 将命令的输出作为普通的shell命令执行

ctrl+r/s 搜索命令行历史

  • ctrl+r 反向搜索
  • ctrl+s 前向搜索，回车执行搜索到的命令

    ctrl+s 可能和xon/xoff冲突，可以通过stty -ixon禁用

at

  • contab 周期性定时任务，at一次性任务
  • at now + 1 hour，回车输入命令，ctrl+d完成
  • atq查看工作序列
  • atrm 删除工作序列

按功能搜索命令

  • apropos + 想要干的事情

    e.g. apropos build filesystem

管理系统版本的替代系统

  • 路径 /etc/alternatives 符号链接到该目录

  • debian:

    update-alternatives

  • centos:

    alternatives

彻底删除文件

  • shred: rm命令的升级版，使用shred删除文件后，数据块会被随机复写

执行完命令后返回当前目录

  • 子进程方式: (cd /etc/ && ls -l)
