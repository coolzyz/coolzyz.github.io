环境

  • setfenv(set function environment)改变一个函数的环境，该环境参数是一个函数层级和一个table，函数层级1表示当前函数，2表示调用当前函数的函数，以此类推

  • 由于函数继承创建其函数的环境，所以一个程序块改变了自己的环境，后续由它创建的函数都共享新环境
